## TECHNICAL NOTES
- Use CSS transitions for animations (NO Framer Motion for MVP)
- transition-all duration-200 for hover states
- Use Inter from next/font/google
- Use lucide-react for icons
- Use shadcn Button, Input, Card, Dialog, Sheet, Badge components
- Add shadcn components via: npx shadcn add button input card dialog sheet badge select label separator
- All demo data should be clearly marked with purple "DEMO" badges
- Respect prefers-reduced-motion
- Make it fully responsive (mobile-first)
- Use the custom color palette consistently throughout

## IMPORTANT
- Do NOT use a gradient mesh background — plain #0A0A0F
- Do NOT add fake social proof counters
- The "Simulate a Failure" button is the primary conversion element — make it prominent
- All buttons should have hover states (scale 0.98 on press, lift on hover)
- Cards should lift slightly on hover

Build ALL of these components. Make sure the landing page looks stunning and modern. The dashboard should feel premium but functional.
